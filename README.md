A hybrid game combining classic Tetris mechanics with 2048-style tile merging logic.

Features
7 tetromino shapes
Ghost piece
Hold system
Difficulty scaling
Chain merging system
Gravity handling
Persistent leaderboard
Sound effects
Pause/resume system
Technologies
Python
NumPy
Pygame
Object-Oriented Programming
Algorithms
BFS-based connected component detection
Wall-kick rotation system
Chain merge mechanics
Gravity compaction algorithm
Installation
pip install -r requirements.txt
python Tetris_2048.py
